Developer Documentation for the Flint System

Overview

The Flint source code contains all the files you need to run a analysis job using. Note that the Flint system is specifically designed to analyze metagenomics whole-genome sequencing data.

Flint Metagenomics

Flint is a metagenomics profiling pipeline that is built on top of the Apache Spark framework, and is designed for fast real-time profiling of metagenomic samples against a large collection of reference genomes.
